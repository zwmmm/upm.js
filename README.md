# upm.js

> 一个方便管理 umd 格式的 cli

![image](http://qed4xvaoy.bkt.clouddn.com/upm.gif)

## 下载

```bash
npm i upm.js -g
```

## 使用

如果有七牛云，则可以先配置七牛云
`upm init`

仅仅只是下载
`upm install <name>`
比如我要下载vue的umd版本文件
`upm install vue`

其他的命令
1. `upm config list`
2. `upm config set <key> <value>`
